# Factorial

`factorial(n)` multiplica todos los enteros positivos desde `1` hasta `n`.

Por convención, `factorial(0)` devuelve `1`. Lo apunto porque es justo el tipo
de caso pequeño que yo olvidaría al simplificar un bucle con demasiada confianza.

La función se usa en ejemplos de diagnóstico porque tiene un caso base claro y
un comportamiento fácil de comprobar con tests pequeños.
